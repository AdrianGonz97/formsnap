Exposes the following additional values from superforms as slot props via the `<Form.Root/>` component. 

- `allErrors`
- `delayed`
- `errors`
- `submitting`
- `tainted`
- `timeout`
- `validate`
- `posted`
- `fields`
- `message`
- `options`
- `formId`
- `restore`
- `capture`
- `constraints`
